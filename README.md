# Digicheck

複数のURLに含まれる `recipe=` パラメータを解析し、カードごとの合計枚数を集計するWebツールです。

## 機能

- 最大3つのURLまたはrecipeパラメータ文字列を入力
- カード名データと合計枚数を一覧表示
- 合計枚数が5枚以上のカードを別途ハイライト表示
- すべてブラウザ内で処理（サーバー送信なし）

## ファイル構成

```
index.html   - 入力ページ
result.html  - 結果表示ページ
style.css    - スタイルシート
```

## GitHub Pages での公開方法

1. このリポジトリを GitHub にプッシュ
2. リポジトリの **Settings** → **Pages** を開く
3. **Branch** を `main`、フォルダを `/ (root)` に設定して **Save**
4. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセス可能になります
